How to Typescript?

Typescript is supported by your favorite programmer's editor of choice


  • Visual Studio

  • Sublime Text

  • Brackets

  • Vim

  • Jetbrains' Webstorm

  • Visual Studio Code (aka VS Code)

VS Code is a new editor recently released by Microsoft. It has excellent built in support for Typescript, ASP.Net 5, NodeJs, and integrated Git support.
